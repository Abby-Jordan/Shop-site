# Shree Mahakali Dairy & Sweets

**Shree Mahakali Dairy & Sweets** is a modern, responsive landing page for a premier dairy and confectionery business. It blends a clean, premium aesthetic with practical features to drive customer engagement and orders.

## 🚀 Key Features

- **Product Showcase**: A dynamic, categorizable gallery showcasing fresh milk, ghee, paneer, and sweets.
- **WhatsApp Integration**: Instant "Order on WhatsApp" buttons across the site for direct customer-to-business sales.
- **Responsive Design**: Mobile-first architecture ensures a seamless experience on all devices.
- **Premium UI**: Smooth animations and a polished interface that reflects the purity of the brand.

## 🛠️ Tech Stack

- **Framework**: React 19 (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Components**: Radix UI

## 📦 Quick Start

1. **Install**: `pnpm install`
2. **Dev**: `pnpm run dev`
3. **Build**: `pnpm run build`
